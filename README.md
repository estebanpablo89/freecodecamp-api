# Freecodecamp api challenge

> Micro API for freecodecamp challenge

## Demo

The API is live at [https://freecodeapi-estebanpablo89.herokuapp.com/](https://freecodeapi-estebanpablo89.herokuapp.com/)

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start

```
